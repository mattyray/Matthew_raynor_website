My Development Projects

I'm an AI Engineer & Full-Stack Developer who builds production-grade software from the ground up. My work spans computer vision, LLM integration, agentic AI systems, and full-stack web development using React, TypeScript, Django, and modern cloud tooling.

I specialize in taking complex real-world processes and transforming them into clean, reliable, user-friendly software. My engineering approach is methodical — I plan deeply, break down problems clearly, and design systems that scale.

Recent work includes an agentic AI shopping assistant using LangChain and Claude that manages carts and searches photos semantically — try it live. Also: an enterprise computer vision pipeline with TensorFlow.js and MediaPipe for real-time photo validation, and ML-powered wall visualization using MiDaS depth estimation. I also build full-stack platforms like ToteTaxi.com with complex pricing engines and third-party API orchestration.

Featured Projects

ToteTaxi.com

A full-stack logistics platform for Manhattan-to-Hamptons delivery services. Features a multi-step booking wizard with dynamic pricing engine, dual user models, and real-time delivery tracking via Onfleet API. Built with React 19, TypeScript, and Django REST Framework — includes role-based dashboards, Stripe payment processing, and complex third-party API orchestration.

View Project

IDP EasyCapture

An enterprise computer vision system built for IDP Americas, a multinational corporation. Uses TensorFlow.js and MediaPipe for real-time image quality validation — detecting blur, lighting issues, face positioning, and document alignment before submission. Built with Next.js, Django REST Framework, and deployed as a production pipeline processing thousands of photos.

View Project

HistoryFace

An AI SaaS app that transforms selfies into historical figures using the face_recognition library (dlib) for facial landmark detection and HuggingFace diffusion models for image generation. Features intelligent face swapping, a freemium business model with Stripe, and optimized inference for cost management at scale.

View Project

EJArtMover.net

A complete business management system for a fine art moving company based on Long Island, NY. Features work order tracking, client management, an interactive FullCalendar dashboard with color-coded job scheduling, PDF invoice generation, and AJAX-powered forms. Built with Django, PostgreSQL, Bootstrap 5, and WeasyPrint — deployed via Docker on Heroku.
