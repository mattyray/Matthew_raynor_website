My Journey

Once a commercial fisherman, I became tetraplegic after a life-altering injury. But I turned pain into purpose, teaching myself to code with a stylus, building apps, and inspiring others through writing and art.

My Development Projects

I'm an AI Engineer & Full-Stack Developer who builds production-grade software from the ground up. My work spans computer vision, image processing, SaaS architecture, workflow automation, and full-stack web development using React, TypeScript, Django, and modern cloud tooling.

I specialize in taking complex real-world processes and transforming them into clean, reliable, user-friendly software. Because I'm paralyzed, I naturally adopted a highly structured and methodical engineering approach — I plan deeply, break down problems clearly, and design systems that scale.

My recent work includes an AI-powered ID photo capture system with on-device ML and computer vision, a full logistics operations platform for ToteTaxi.com, and a viral AI face-swapping SaaS combining facial recognition with GPU-accelerated image generation.

Featured Projects

ToteTaxi.com

A full-featured logistics platform built for a company that transports clients from Manhattan to their second homes in the Hamptons. Features a multi-step booking wizard, dual user models for clients and staff, and real-time delivery integration via Onfleet. Built with React 19, TypeScript, and Django REST Framework — featuring role-based dashboards, order management, Stripe integration for payment processing, and seamless third-party API orchestration.

IDP EasyCapture

An enterprise AI-powered photo capture and validation system built for IDP Americas, a multinational corporation headquartered in Rhode Island. Uses machine learning to detect image quality issues in real-time, ensuring documents meet compliance standards before submission. Built with Next.js, Django REST Framework, and HuggingFace AI integration.

HistoryFace

An AI SaaS app that transforms selfies into historical figures using facial recognition and HuggingFace machine learning models. Features a freemium business model with Stripe integration, real-time processing, and smart cost management.

Features work order tracking, client management, an interactive FullCalendar dashboard with color-coded job scheduling, PDF invoice generation, and AJAX-powered forms.
