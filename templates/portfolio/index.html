My Portfolio & Story

From rebuilding my life after a spinal cord injury to building production-grade software—this is how I turn resilience into real-world code.

What I Offer

Whether you're a business owner in need of a custom web application or a company looking for a passionate developer, I build software that works in the real world.

I'm Matthew Raynor—a former commercial fisherman turned full-stack web developer after a life-changing spinal cord injury. I taught myself how to code without the use of my hands, learning Django and Python from scratch with a stylus and sheer determination. That same determination drives every project I build today.

I've developed a full-featured logistics and invoicing system for an art moving company, complete with client management, calendar-based job scheduling, PDF invoicing, and drag-and-drop UI. I've built an e-commerce store, a donation platform with blog and video integration, and an AI chatbot using OpenAI's API for real-time support.

For clients, I create secure, scalable, and easy-to-use web apps tailored to your workflow. For employers, I bring production-level skills in Django, Docker, PostgreSQL, React, and a lived perspective on problem-solving, perseverance, and accessibility. I build with both code and compassion—and I'm ready to add value from day one.
